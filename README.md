# The Dojo Blog
这是一个用React搭建的博客小项目.


source:https://github.com/iamshaunjp/Complete-React-Tutorial/tree/main


实现了Home、Navbar、Create、BlogList、BlogDetails等组件。
使用json-server模块模拟后端接口。

## 下载依赖
```
npm install
```
## 模拟后端服务器
```
npx json-server --watch data/db.json --port 8000
```

## 启动项目
```
npm run start
```
